Installation   
To successfully run the project, complete the steps below.
1.  **Clone the repository**
```bash
git clone https://github.com/Water-Shadow23/Momentia.git
```
2. **Install Dependancies**
 
   *  **Backend**   
   ```bash 
   cd Server
   npm i
   ```      
   * **Frontend**
   ```bash
   cd Client
   npm i
   ```
3. **Run locally**

   * **Server**   
   ```bash 
   cd Server
   npm start
   ```  
   * **Client**
   ```bash
   cd Client
   npm run dev
   ```
