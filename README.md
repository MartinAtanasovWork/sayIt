###How to run the application    

1.Cloning the repo to your computer! 
2. Install Dependancies 
   *  **Client**   
   ```bash 
   cd client
   npm install
   ```      
   * **Server**
   ```bash
   cd Server
   npm install
   ```
3. Startin the application locally
   * First you start the **Server**   
   ```bash 
   cd server
   npm run server
   ```  
   * Then you start the **Client**
   ```bash
   cd client
   npm run client
   ```
